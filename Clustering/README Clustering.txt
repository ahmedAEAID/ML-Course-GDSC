Hierarchical Clustering for Customer Segmentation
• Developed a customer segmentation strategy using hierarchical clustering on a dataset of active credit card holders.
• Utilized 18 behavioral variables to create clusters based on usage behavior.
• Explored different combinations of linkage criteria and distance metrics to find the optimal configuration.
• Evaluated cluster quality using silhouette analysis.
• Incorporated feature reduction techniques such as PCA for improved clustering results.
• Visualized clustering results using dendrograms and T-SNE plots.
• Compared different transformation techniques to enhance clustering performance.
